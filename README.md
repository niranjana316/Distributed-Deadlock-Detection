# Distributed Deadlock Detection using Wait-For Graph

## Objective
To simulate distributed deadlock detection using a Wait-For Graph and edge-chasing algorithm.

## Tools Used
- Python
- SimPy
- Streamlit
- NetworkX

## How it Works
- Processes are represented as nodes
- Dependencies are edges
- Cycle detection = deadlock detection

## How to Run
```bash
streamlit run app.py
